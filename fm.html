Farmacocinético

Criação de um modelo para um sistema farmacocinético e o uso desse modelo para responder uma questão criada. Foi necessário: 1º fazer uma pesquisa para tentar entender melhor o sistema 2º pensar sobre as diferentes questões possíveis para serem investigadas, analisadas e modeladas. 3ºAbstrair, implementar e validar um ou mais modelos para o sistema (feito no python) 4ºProduzir um cartaz de aparência profissional.
