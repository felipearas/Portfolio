Termômetro resistivo

Termômetro feito a mão, medido através de resistência.

Foi utilizado um palito de madeira, onde foi enrolado cerca de 6 metros de fio de cobre, para formar uma resistência.

Os dados coletados foram os valores de resistência do termômetro para cada uma das temperaturas indicadas pelo professor. Para realizar tal coleta, utilizou-se uma ponte de Wheatstone.

Ao realizar a calibração, nós definimos a temperatura com base na leitura dos valores de resistência do termômetro, adotando alguns fatores como possíveis médias e possíveis erros. Utilizando esses dados, foi obtida uma equação, que expressa a temperatura em função da resistência
